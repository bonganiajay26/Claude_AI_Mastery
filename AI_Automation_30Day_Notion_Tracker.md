## How to import into Notion

1. Open Notion → **Settings & Members → Import → Markdown & CSV**.
2. Pick this file: `AI_Automation_30Day_Notion_Tracker.md`.
3. Notion will create a full page with the headings and tables.
4. Click any `•••` on a table → **Turn into database** to get filters, tags, and relations.
5. Recommended database conversions:
   - **Daily Tracker** → database with properties: Week (select), Topic (text), Done (checkbox), Notes (text).
   - **Job Tracker** → database with: Status (select), Applied date (date), Company (text), Role (text).
   - **Interview Prep Log** → database with: Confidence (number), Practiced (checkbox).
6. Create a master dashboard page; embed linked views of each database filtered by "this week" or "in progress."
